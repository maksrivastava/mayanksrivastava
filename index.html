Medical Affiars Text Miner

Worked as a Technical Lead in the project.

The client's SharePoint Site Collection had numerous lists with numerous multiline text fields. The client, for its business analytics purposes, wanted to determine where (which fields) the stakeholders were entering the data more.

I conceptualized the idea into a POC and which was well received by the client. I created a SharePoint page (which I named Text Miner) that would allow the users to search the keywords. This search engine was a Google like search that would not only provide suggestions (populating from the lists) to the users but also a graphical representation of the number of times the keyword is found in each of the fields in the various lists. It also showed the data table with relevant search results.

This solution warded the most innovative solution in the client's annual meet.

Service Request Manager

Worked as SharePoint Lead / Developer

A lot of different software tools and applications are used by the Pharmaceutical client for their various business functions. If the users faced any issues with any of the tools, they would email to the Medical Affairs team and they would assign it to the various vendors maintaining the respective applications. The vendors will then assign the issues to the relevant team members. This complete process was irregular, complex and convoluted to manage and track.

The solution I provided to the above business problem was a one stop shop called the Service Request Manager for all the users to log and report the issues and the Medical Affair teams to track the tickets. The Vendor can then assign the ticket to the team member, emails being triggered at each step.

This solution reduced the service delivery time for all deliverables and gave a detailed as well as a bird eye view to the MA team.

Service Request Manager Dashboard

An interactive and graphical dashboard to show and filter all requests. The Medical Admin Team, at any point of time can view the complete data using filters like the ticket statuses, who the tickets are assigned to, for which tools/applications, etc. and the data can be exported to Excel.

Service Request Form

The link to the Service Request Form was provided on all the tools where the users can report any kind of issues they are facing.

Service Request Form

The form submission and updation would trigger various workflows which would send an email to the respective Medical Team member who would then assign it to the Vendor.
